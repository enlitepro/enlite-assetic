Установка
=========

Для установки необходимо добавить в composer.json следующие строки

```json
{
    "repositories": [
        {
            "type": "composer",
            "url": "http://packages.dev.infonet.by/"
        }
    ],

    "require": {
        "zf2/stdlib-assetic": "1.*"
    },
}

```
